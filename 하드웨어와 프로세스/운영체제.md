# 운영체제란?

## 1. 운영체제 (Operating System, OS)의 정의

운영체제 (OS, Operating System)란 컴퓨터 하드웨어와 사용자(또는 응용 프로그램) 사이에서 중재자 역할을 하는 시스템 소프트웨어이며
사용자가 컴퓨터를 편리하고 효율적으로 사용할 수 있도록 하드웨어 자원을 관리하고, 응용 프로그램이 실행될 수 있는 환경을 제공한다.

---

## 2. 운영체제의 주요 역할

### 1) 프로세스 관리 (Process Management)

- 프로그램 실행 및 종료 관리
- CPU 스케줄링
- 멀티태스킹 지원
- 프로세스 간 통신 (IPC)

### 2) 메모리 관리 (Memory Management)

- 주기억장치 (RAM) 할당 및 해제
- 가상 메모리 관리
- 메모리 보호 기능

### 3) 파일 시스템 관리 (File System Management)

- 파일 생성, 삭제, 수정
- 디렉토리 구조 관리
- 저장 장치 접근 제어

### 4) 입출력 (I/O) 관리

- 키보드, 마우스, 프린터 등 장치 제어
- 장치 드라이버 관리

### 5) 보안 및 사용자 관리

- 사용자 계정 관리
- 접근 권한 제어
- 데이터 보호

---

## 3. 운영체제의 목적

1. **편리성** - 사용자가 쉽게 컴퓨터를 사용할 수 있도록 지원
2. **효율성** - 시스템 자원을 효율적으로 활용
3. **안정성** - 오류 및 충돌 방지
4. **확장성** - 다양한 프로그램과 하드웨어 지원

---

## 4. 운영체제의 종류

### 1) 단일 사용자 운영체제

- 한 명의 사용자만 사용 가능

### 2) 다중 사용자 운영체제

- 여러 사용자가 동시에 사용 가능

### 3) 실시간 운영체제 (RTOS)

- 즉각적인 처리가 필요한 시스템에 사용
- 예: 항공기 제어 시스템, 의료 장비 등

### 4) 네트워크 운영체제

- 네트워크를 통해 자원 공유 지원

---

## 5. 대표적인 운영체제 예시

- Windows
- macOS
- Linux
- Android
- IOS

---

## 6. 운영체제의 구조

운영체제는 응용 프로그램과 하드웨어 사이에서 자원을 관리하고 명령을 전달하는 역할을 수행한다.

사용자
↓
응용 프로그램
↓
운영체제
↓
하드웨어

---

## 7. 정리

운영체제는 컴퓨터 시스템의 핵심 소프트웨어로서 하드웨어 자원을 효율적으로 관리하고 사용자에게
편리한 환경을 제공하는 역할을 한다.
